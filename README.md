# Dashboard_AACT
Dashboard Power BI 100% propio sobre AACT (ClinicalTrials.gov) 426K ensayos clínicos analizados. Total estudios y enrollment, Duración media y % completados/terminados, Evolución anual, Mapa mundial de estudios completados y termination rate por fase y país, filtros cruzados por fase, status, agencia, año y tipo. Todo con DAX propio
